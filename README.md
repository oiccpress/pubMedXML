# PubMed XML

This plugin provides PubMed XML export for articles.

A new route `/jtap/article/viewXML/11207` will be provided.

Note you will need to edit your theme to surface a button to this i.e

```
<a href="{url page="article" op="viewXML" path=$article->getBestId()}">XML</a>
```
